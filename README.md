Projektbeschreibung

Dieses Projekt ist eine React-Webanwendung, die ein einfaches Registrierungsformular mit Live-Validierung umsetzt.
Der Fokus liegt auf dem Lernen von React-Grundlagen, insbesondere:

React Functional Components

useState für Formular-State

Echtzeit-Validierung für:

Benutzername (Vor- und Nachname)

E-Mail-Adresse

Handynummer

Layout mit CSS & Flexbox

Strukturierte Komponenten (Header, Body)

Fehlermeldungen werden dynamisch angezeigt und verschwinden automatisch, sobald die Eingaben korrekt sind.
Das Projekt dient als Lern- und Übungsprojekt, um den Umstieg von klassischem JavaScript (DOM-Manipulation) auf React-State-basiertes Arbeiten zu verstehen.

Technologien

React 

JavaScript 

CSS / Flexbox

Regex für Formularvalidierung

Ziel des Projekts

Ziel ist es, die Grundlagen von React zu festigen und ein sauberes, benutzerfreundliches Formular zu entwickeln, das später erweitert werden kann .
